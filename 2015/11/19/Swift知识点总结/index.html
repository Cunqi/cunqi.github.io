Swift知识点总结

Posted on 2015-11-19   |   In Programming

#写在前面
这是我在阅读The Swift Programming Language时做的笔记，主要记录的是一些我没办法立刻记住的知识点，以供查阅。

let与var的区别

  • let 为常量，一旦赋值后不可改变, var为变量，赋值后可以改变其引用

分号

  • 分号在Swift中并不强制要求使用，但是如果在一行代码内编写多条独立语句则需要使用分号

类型别名

  • 使用类型别名来给一个类型取一个更有意义的名字

元组

  • 元组可以使用下标访问，类似数组，从0开始

可选类型（Optional）

  • 对一个被声明的可选变量或常量没有赋值，他们会被自动设置为nil
  • 可以使用if 语句来进行nil的比较
  • 使用!进行强制解析，表明确定该可选类型必定有值
  • 使用类似 if let x = str? 的语句完成可选绑定
  • 使用!声明一个隐式可选类型

错误处理

  • 使用 do {} catch {}来检查一个错误

使用断言进行调试

  • assert(x >= 0, “error message”)
  • assert(x >= 0)

赋值操作

  • 赋值操作不会有返回值

取余运算

  • 在对负数取余时，会忽略除数的符号，这意味着a % b 等价于 a % -b
  • Swift 可以对浮点数取余 8 % 2.5 = 0.5

空合运算符

  • 空合运算符作用于可选类型
  • a ?? b 表示 先取可选类型a的值，如果a的值为nil，那么返回默认值b
  • a的原始值类型必须和b的相同
  • 空合运算符也使用了短路运算符

区间运算

  • 闭区间运算(1…5) 3个点， 返回[1, 2, 3, 4, 5]
  • 半开区间运算(1..5) 2个点， 返回[1, 2, 3, 4]

字符串

  • 字符串是值类型，猜想在赋值的时候是复制了字符串的值，使得每个变量的值都相互独立
  • Character 和 String 都使用 ""双引号赋值
  • 字符串类型兼容Unicode标准
  • 字符串类型的下标语法使用的是String.Index类型
  • insert方法插入的是一个Character， insertContentsOf方法插入的是一个String

集合类型

数组

  • 数组也可以使用+=来添加相同类型的数组
  • 数组下标使用数字进行访问num[0]

集合

  • 集合对象可以使用sort()返回一个有序的集合
  • 集合支持常用的操作， 交，并，减，加

字典

  • 在使用空字面量创建新集合时，需要预先知道该字典的类型。
  • 通过下标对字典的某个键值对赋值为nil移除该键值对

Switch语句

  • Swift中Swift语句不存在隐式贯穿，即对于每个case不需要写break
  • 支持区间匹配
  • 支持多值匹配
  • 使用where语句为每个case添加限制条件

检测API可用性

函数

  • 使用元组完成多个返回值的返回
  • 将拥有默认参数值的变量放在函数签名的最后（如果有多个参数都拥有默认参数值怎么办？）
  • 在类型后面添加...则讲变量声明为可变参数，同理可变参数需放在函数签名的最后
  • 一个函数最多拥有一个可变参数
  • 对参数名称使用var关键字，则将该变量变为可修改，但依然只是一个值副本的拷贝
  • 变量参数仅仅存在于函数调用的生命周期中。
  • 使用&表示这是一个输入输出参数，且在函数签名中需使用inout关键字
  • 函数也有类型，函数的类型即为它的函数签名，即参数列表 -> 返回值
  • 函数也可以通过赋值语句指定给变量
  • 函数也可以作为返回值
  • 函数定义在另一个函数体中被称为嵌套函数，而该函数体则被称为外围函数

闭包

  • 内联闭包表达式构造的闭包作为参数传递给函数或方法时,都可以推断出闭包的参数和 返回值类型
  • 参数名称缩写，使用$ + 数字 完成对闭包参数的调用，$0开始
  • 自动闭包让你能够延迟求值

枚举

  • 关联值：枚举类型可以存储的值
  • 原始值：在定义时就预先定义好的值
  • 当原始值为字符串类型时，值为每个枚举类型的字符串表示

类与结构体

  • 类与结构体大部分相同
  • 类可以允许继承，可以被析构，可以被引用计数
  • 结构体通过复制在代码中传递，不存在引用计数
  • 结构体拥有一个成员逐一构造器，新实例中各个属性的初始值可以通过属性的名称传递到成员逐一构造器之中。
  • 结构体和枚举类型是值类型（值类型被赋予给一个变量、常量或者被传递给一个函数的时候,其值会被拷贝。）
  • 类是引用类型

字符串(String)、数组(Array)、和字典(Dictionary)类型的赋值与复制行为

  • String, Array, Dictionary都使用结构体实现，这意味着在赋值时使用的是值拷贝
  • NSString, NSArray, `NSDictionary·则使用的类实现，这意味着在赋值时使用的是引用传递
  • 值类型在声明为常量后，其属性不允许被修改
  • 引用类型则相反
  • 延迟存储属性：使用lazy关键字，使得属性在第一次调用的时候才会被赋值

计算属性

  • 通过提供getter 和 setter方法来间接获取或设置其他属性

类

  • Swift中不存在隐式基类(如Java中的Object)
  • 使用override关键字表示重写
  • 使用final关键字防止重写

构造器

  • 构造器为属性赋值，不会触发属性观察器
  • 便利构造器来调用同一个类中的指定构造器
  • 指定构造器必须总是向上代理
  • 便利构造器必须总是横向代理

拓展

  • extension 可以对一个已有类型添加新的功能，但是不能覆写(override)已有功能
  • 拓展可以添加计算属性，但是不能添加存储属性和属性观察器

模块和源文件

  • 模块指的是以独立单元构建和发布的 Framework 或 Application
  • 源文件指的是 Swift 中的 ,就是编写 Swift 源代码的文件
  • 公共级别的API需要是用public 来修饰，否则默认的internal级别已经够了
